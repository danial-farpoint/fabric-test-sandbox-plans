Dequeue Optimization: Sequential vs Concurrent Spawning

BEFORE: Sequential Await
1 await spawnDiscoveryChild(task1)
2 await spawnDiscoveryChild(task2)
3 await spawnDiscoveryChild(task3)
! 3 slots free, but 3× startup latency
AFTER: Fire-and-Forget
1 spawnDiscoveryChild(task1).catch(...)
2 spawnDiscoveryChild(task2).catch(...)
3 spawnDiscoveryChild(task3).catch(...)
✓ All 3 spawn concurrently — single startup latency
